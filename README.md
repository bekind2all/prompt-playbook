# 🌟 Prompt Playbook
AI 활용 강의를 진행하며 실제 현장에서 사용한 프롬프트를 정리한 모음집입니다.  
기업, 공공기관, 교육기관에서 바로 활용할 수 있도록 분야별로 구조화하였습니다.

---

## 📚 About This Repository
이 저장소는 다음을 목표로 합니다.

- 실무에서 바로 사용할 수 있는 **프롬프트 템플릿 제공**
- 기업·기관 대상 강의에서 활용한 **사례 기반 프롬프트 공유**
- 강의, 교육, 워크숍에서 재사용 가능한 **표준형 프롬프트 구조화**
- 생성형 AI를 처음 사용하는 사람도 쉽게 적용할 수 있도록 **체계화된 가이드 제공**
