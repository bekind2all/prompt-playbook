📝 Universal Blog Writing Prompt

블로그 글을 빠르고 완성도 높게 작성할 수 있도록 만든 범용 프롬프트 템플릿입니다.
사용자는 단순히 [ ] 안을 채워 넣기만 하면 고품질 블로그 글을 얻을 수 있습니다.

⸻

📌 1. 작성 목적 (Purpose)

블로그 글의 방향을 명확히 하기 위해 아래 항목을 입력하세요.
	•	글 작성 목적: [예: 제품 소개, 리뷰, 정보 공유, SEO 상위노출 등]
	•	타깃 독자: [예: 2030 직장인, AI 관심자, 부모 독자 등]
	•	핵심 키워드: [예: 챗지피티강사, 프롬프트강사, OO맛집 등]
	•	글의 톤 & 스타일: [친절하게 / 전문적으로 / 가볍게 / 유머러스하게 등]

⸻

📌 2. 사용자 입력값 (User Input Fields)
	•	주제(Topic): [블로그 글 주제 입력]
	•	핵심 메시지: [글에서 전달할 핵심 메시지]
	•	반드시 포함할 경험/사례: [개인 경험 또는 사례]
	•	강조 포인트: [중요 포인트 2~5개]
	•	원하는 글 길이: [2000자 / 3000자 / 5000자 등]
	•	이미지 ALT 텍스트용 설명: [이미지 요약 설명]
	•	내부 링크 여부: [연결할 다른 글 또는 주제]

⸻

📌 3. 출력 구조 (Output Structure)

아래 구조에 맞게 GPT가 출력하도록 설계되어 있습니다.

1. 메타디스크립션 (120~150자)
2. 블로그 제목 (SEO 키워드 포함)
3. 인트로(후킹 문장)
4. 본문
   - 소제목1 + 내용
   - 소제목2 + 내용
   - 소제목3 + 내용
5. 결론(요약 + CTA 또는 공감)
6. 이미지 ALT 태그 목록
7. 내부 링크 추천
8. 해시태그 추천 10개


⸻

📌 4. 블로그 생성 프롬프트 (Copy & Paste)

아래 내용을 그대로 붙여넣어 사용하세요.

⸻

🚀 Prompt

다음 조건을 충족하는 고품질 블로그 글을 작성해줘.

[1] 작성 목적
- 글 목적: [블로그 글 목적]
- 타깃 독자: [타깃 독자]
- 핵심 키워드: [키워드]
- 톤 & 스타일: [지향하는 문체]

[2] 콘텐츠 입력값
- 주제: [작성할 글의 주제]
- 핵심 메시지: [전하고 싶은 핵심 메시지]
- 포함할 개인 경험·사례: [경험 또는 사례]
- 강조 포인트: [강조 포인트 2~5개]
- 글 길이: [원하는 글자 수]

[3] SEO 요구사항
- 제목, 소제목, 본문에 키워드 자연스럽게 포함
- 메타디스크립션 120~150자
- 문단은 짧고 가독성 높게 구성
- 결론은 CTA(행동 유도) 또는 공감형 메시지 포함
- 태그 10개 이상 추천

[4] 이미지 관련
- 이미지 ALT 텍스트 5개 생성 (이미지는 생성하지 않음)

[5] 출력 형식
아래 형식을 반드시 그대로 지켜줘.

1) 메타디스크립션  
2) 블로그 제목  
3) 인트로(후킹 문장)  
4) 본문 (소제목 + 본문 구조)  
5) 결론  
6) 이미지 ALT 텍스트 목록  
7) 내부 링크 추천  
8) 해시태그 10개 이상


⸻

📌 5. 확장 옵션 (Optional)

필요 시 아래 옵션을 추가로 붙여넣을 수 있습니다.
	•	[옵션] 글 한눈에 보기용 카드뉴스 텍스트 생성
	•	[옵션] 인스타그램 캡션 버전 생성
	•	[옵션] 15초 쇼츠 스크립트 생성
	•	[옵션] 예시 3개 추가
	•	[옵션] 표 1개 포함
