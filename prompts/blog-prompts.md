# 📄 **blog-prompts**

👉 이 내용을 그대로 깃허브 파일에 붙여 넣으세요!

```md
# 📝 Blog Writing Prompts  
블로그 글을 빠르게, 그리고 상위노출 전략에 맞춰 작성할 수 있도록 구성한 프롬프트 모음입니다.  
키워드 분석, 제목 생성, 본문 구성, SEO 최적화까지 자동으로 처리할 수 있습니다.

---

## ✨ 1. 네이버 블로그 상위노출용 글쓰기 프롬프트
```

당신은 네이버 블로그 SEO 전문가입니다.
아래 키워드로 상위 노출을 목표로 블로그 글을 작성해 주세요.

📌 필수 조건

* 제목 3개 제안
* 서론 / 본론 / 결론 구조
* 한 문단은 3~4줄로 구성
* 키워드 자연스럽게 반복
* 메타디스크립션 작성
* 이미지 ALT 텍스트 5개 제공
* 독자가 오래 머물도록 체류시간 증가 전략 반영

키워드: "챗지피티강사", "프롬프트강사"

```

---

## ✨ 2. SNS 바이럴 확장형 블로그 글 프롬프트
```

다음 주제를 블로그 + SNS(인스타그램, 유튜브 쇼츠)에서도 활용할 수 있도록
후킹 요소가 강한 콘텐츠로 작성해 주세요.

요구사항

* 강렬한 시작 문장
* 이야기(스토리텔링) 기반으로 구성
* 블로그 버전 + SNS 버전 각각 따로 작성
* 공감 유발 포인트 3개 포함
* 댓글 유도 문장 추가

주제: 사용자가 입력

```

---

## ✨ 3. 전문가 인터뷰형 블로그 글 생성 프롬프트
```

당신은 콘텐츠 작가입니다.
특정 주제에 대해 전문가와 인터뷰한 것처럼
Q&A 형식의 블로그 글을 작성해 주세요.

요구사항

* 질문 5개 생성
* 전문가 답변 형식으로 작성
* 핵심 인사이트 요약 섹션 포함
* 썸네일용 이미지 문구 추천

주제: 사용자가 입력

```

---

## ✨ 4. 교육/강의 후기 작성 프롬프트
```

아래 강의/교육 내용을 바탕으로
독자 친화적인 후기형 블로그 글을 작성해 주세요.

요구사항

* 강의 핵심 내용 요약
* 참여자 반응 3개 생성
* 실제 현장감을 주는 표현 사용
* 마지막에 CTA(Call-To-Action) 포함
  (예: 교육 문의 안내)

강의명: 사용자가 입력
내용 요약: 사용자가 입력

```

---

## ✨ 5. 제품/서비스 리뷰 글쓰기 프롬프트
```

블로그 리뷰 전문가 역할을 해 주세요.
다음 정보를 기반으로 리뷰형 포스팅을 작성해 주세요.

요구사항

* 장점 5개 / 단점 2개
* 직접 사용한 것처럼 상세 묘사
* 이미지 ALT 5개
* 결론에 한줄 요약 포함
* 해시태그 10개 추천

제품/서비스 정보: 사용자가 입력

```

## ✨ 6. 목차 기반 확장형 글쓰기 프롬프트
```

다음 키워드로 블로그 글의 전체 목차 6개를 먼저 만들고,
각 목차를 3단락씩 확장하여 본문을 작성해 주세요.
모든 문단은 3~4줄로 구성합니다.

키워드: 사용자가 입력

```

## ✨ 7. 글쓰기 스타일 변경(톤 & 매너 변환) 프롬프트
```

아래 글을 주면 다음 5가지 스타일로 각각 변환해 주세요.

1. 교사·교육자 스타일
2. 기자·칼럼니스트 스타일
3. 유튜브 쇼츠 대본 스타일
4. 전문가 보고서 스타일
5. 친근한 SNS 스타일

본문: 사용자가 입력

```

## 📌 활용 Tip
- 모든 프롬프트는 **사용자의 목적에 맞게 수정 가능**  
- 강의 커리큘럼, 기관 요청사항, 글쓰기 수준별로  
  난이도 조절도 쉽게 할 수 있음  
- 필요하다면 “자동 이미지 생성용 프롬프트”도 추가 가능
