# 📧 Email Writing Prompts  
기업·기관·학교·외부 업체와의 소통에 즉시 활용 가능한 이메일 작성 프롬프트 모음입니다.  
정중한 표현, 핵심 요약, 요청사항 리마인드, 마무리 멘트까지 자동으로 정리할 수 있습니다.

---

## ✨ 1. 기본 비즈니스 이메일 자동 작성 프롬프트
```

당신은 전문 비서입니다.
아래 내용을 기반으로 깔끔하고 정중한 비즈니스 이메일을 작성해 주세요.

📌 요구사항

* 제목 포함
* 핵심 내용 5줄 내 요약
* 요청사항을 마지막에 한 번 더 강조
* 지나친 딱딱함은 피하고, 자연스럽고 정중하게
* 마무리 인사 + 서명 포함

[이메일 내용]
<< 여기에 내가 쓰고 싶은 내용을 입력 >>

```

---

## ✨ 2. 기관·공공기관 업무 협조 요청 이메일
```

당신은 공공기관 업무 담당자입니다.
아래 요청 내용을 기반으로 공손하고 정확한 협조 요청 이메일을 작성해 주세요.

요구사항

* 공문 스타일의 깔끔한 구조
* 요청 배경 → 요청 내용 → 일정 → 필요 시 회신 요청
* 담당자 입장에서 이해하기 쉽게 작성
* 첨부파일 안내 문구 포함

[요청 내용]

```

---

## ✨ 3. 교육·강의 제안 이메일 (강사 버전)
```

당신은 기업/기관을 대상으로 강의를 진행하는 전문 강사입니다.
아래 정보를 바탕으로 상대 기관에 강의 제안을 보내는 이메일을 작성해 주세요.

요구사항

* 제목 3개 제안
* 강의 개요 + 기대 효과 정리
* 강사 소개 3줄
* 가능한 일정 2~3개 제안
* 필요 시 회신 요청 문구 포함
* 친절하지만 전문가다운 톤

[강의 주제]
[제안 이유]

```

---

## ✨ 4. 일정 조율·미팅 잡는 이메일
```

당신은 전문 커뮤니케이션 매니저입니다.
아래 내용을 기반으로 일정 조율용 이메일을 작성해 주세요.

요구사항

* 후보 일정 3개 제시
* 온라인/오프라인 여부 확인
* 필요 시 자료 사전 공유 요청
* 상대가 편한 시간으로 다시 제안할 수 있게 문구 포함

[조율할 내용]

```

---

## ✨ 5. 외부 업체에 견적/자료 요청하는 이메일
```

당신은 기업 마케팅 담당자입니다.
아래 목적에 따라 외부 업체에게 견적 또는 자료를 요청하는 이메일을 작성해 주세요.

요구사항

* 요구사항 리스트화
* 제공받을 구체적 자료 명시
* 제출 기한 안내
* 연락 가능한 방법 정리
* 상대의 수고를 배려하는 문구 포함

[요청 내용]

```

## ✨ 6. 강의 후 감사 인사 이메일 (교육 종료 후)
```

당신은 강의를 마친 후 기관 담당자에게 감사 인사를 보내는 강사입니다.
다음 정보를 바탕으로 감사 이메일을 작성해 주세요.

요구사항

* 강의에 참여한 대상의 반응 요약
* 함께 협조해준 담당자에 대한 감사 표시
* 차기 교육 제안 자연스럽게 포함
* 자료 전달 시 첨부 파일 설명 포함

[강의명]
[진행 날짜]

```

## ✨ 7. 민원·불편 사항 대응 이메일 (온화한 버전)
```

당신은 고객 응대 전문가입니다.
아래 불편 사항을 기반으로 진정성 있는 사과 + 해결 방안을 담은 이메일을 작성해 주세요.

요구사항

* 사과 → 공감 → 해결 방안 → 후속 조치 순서
* 문제를 반복하지 않겠다는 의지 표현
* 고객이 느낄 불편을 최소화하는 문구 사용
* 담당자 연락처 안내

[민원 내용]

```

## ✨ 8. 회의록·정리본 공유 이메일
```

다음 회의 내용을 기반으로 회의록을 작성하고, 이메일 본문에 요약 형태로 정리해 주세요.

요구사항

* 회의 목적
* 주요 논의 3~5개
* 결정 사항
* 다음 액션 아이템 3개
* 담당자 및 일정 정리
* 회의록 첨부 안내

[회의 내용 요약]

```

## 📌 활용 Tip
- 동일한 프롬프트라도 **직업·기관 유형(기업/학교/공공기관)에 맞게 톤 조정** 가능  
- 자주 쓰는 이메일은 개인 템플릿으로 저장 후 반복 사용  
- 강의 문의·미팅 일정·보고 요청 등 모든 상황에 쉽게 변형 가능
